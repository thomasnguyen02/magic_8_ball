# Magic 8 Ball 🎱

A simple Python program that simulates a Magic 8 Ball by generating random responses to a user’s question.

## Features
- Generates random responses
- Simple and interactive
- Uses conditional logic

## How it works
The program generates a random number and assigns a response based on that number.

## What I learned
- Python variables
- Conditional statements (if / elif / else)
- Random number generation

## Example Output
Question: Will I become a profitable trader this year? 
Answer: Yes - definitely

## Note
- This was one of my first projects and helped me understand how control flow works in Python.
- This project currently uses predefined variables, and I plan to expand it by adding user input to make it more interactive.
